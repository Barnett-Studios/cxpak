# v2.1.0 Manual Browser Smoke Test

**Spec reference:** Success Criterion 11 in `docs/plans/2026-04-17-v210-design.md`.

**Purpose:** The automated suite (Criteria 1–10) cannot observe runtime DOM behavior. This manual smoke gate verifies the SPA actually works in real browsers. A release MUST NOT be tagged on any failure.

## Generating the artifact

```bash
RUSTUP_TOOLCHAIN=1.94.1 cargo build --all-features --release
./target/release/cxpak visual --visual-type all --format html --out cxpak-all.html .
```

The resulting `cxpak-all.html` is a single self-contained file (~1.6 MB on the cxpak repo itself).

## 8-step checklist

Run each step in each cell of the matrix above. Mark ✅ pass or ❌ fail per browser.

### 1. Self-contained — zero network requests

Open DevTools → Network panel. Reload the page. The Network list must be empty (or only show the page itself). No fetches to CDNs (cdn.jsdelivr.net, unpkg.com, cdnjs.cloudflare.com) or external domains.

### 2. Zero console errors

DevTools → Console panel. Should be clean. Warnings are OK (e.g., autocomplete inputs, deprecation notices). Errors (red) are a failure.

### 3. All 6 nav tabs switch views without reload

Click Dashboard → Architecture → Risk → Flow → Timeline → Diff. Each should switch the visible section without a full page reload (URL hash should change: `#dashboard`, `#architecture`, etc). The browser refresh spinner should not appear.

### 4. Command palette works

Press `Cmd+K` (macOS) or `Ctrl+K` (Linux/Windows). Palette opens with an input focused. Type `health`. The filtered results should include at least one entry matching "health" (view, file, or symbol). Click the result. The view navigates to the corresponding target and the palette closes.

### 5. Theme toggle + localStorage persistence

Click the ☀/☾ theme toggle button in the header. Colors should flip between dark and light. Reload the page (Cmd+R). The chosen theme must persist.

### 6. Inspector opens on node click

Navigate to the Architecture or Risk view. Click any node. The inspector panel should slide in from the right with metrics (PageRank, Risk score, Tokens). Press Esc or click the × to close it.

### 7. Help overlay on `?`

With no palette or inspector open, press `?`. A help overlay should appear listing keyboard shortcuts. Press Esc to dismiss.

### 8. Cross-session persistence

Close the browser tab. Reopen `cxpak-all.html`. The theme chosen in step 5 must still be applied on load (before any user interaction).
